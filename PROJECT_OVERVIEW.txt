## ğŸ”‘ Key Features

### 1. Soulbound Profile NFTs
- One profile per wallet address
- Non-transferable (can't be sold or given away)
- Stores: name, age, interests, photo URL
- Contract: `ProfileNFT.sol`

### 2. Decentralized Matching
- Express interest in other users
- Automatic match creation when mutual
- All matching logic on-chain
- Contract: `Matching.sol`

### 3. Optional USDC Staking
- Stake USDC when planning a date
- Both confirm: 100% refund (minus 5% platform fee)
- One confirms: 150% to confirmer, 50% to no-show
- Neither confirms: 100% refund to both
- 48-hour confirmation window
- Contract: `Staking.sol`

### 4. On-Chain Reputation
- Average star rating (1-5 stars)
- Total dates completed
- Number of no-shows
- All publicly viewable
- Contract: `Reputation.sol`

### 5. Achievement NFTs
- Earn badges for milestones
- "First Date", "5 Star Rating", "10 Dates", etc.
- Collectible NFTs in your wallet
- Contract: `Achievement.sol`

## ğŸ“ Smart Contract Functions

### ProfileNFT
- `createProfile(name, age, interests, photoUrl)` - Create profile
- `updateProfile(...)` - Update existing profile
- `getProfile(address)` - View profile

### Matching
- `expressInterest(targetUser)` - Like someone
- `getMatches(user)` - Get all matches
- `isMatched(user1, user2)` - Check if matched

### Staking
- `createStake(matchedUser, amount, meetingTime)` - Create stake
- `confirmMeeting(matchedUser)` - Confirm you showed up
- `claimStake(matchedUser)` - Claim after window

### Reputation
- `rateUser(user, rating)` - Rate 1-5 stars
- `getReputation(user)` - View reputation
- `getAverageRating(user)` - Calculate average

### Achievement
- `mintAchievement(user, achievementType)` - Award badge
- `getAchievements(user)` - View all badges
